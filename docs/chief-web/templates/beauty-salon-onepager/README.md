# Beauty Salon Onepager Template

Status: Spezifikation v1.0  
Datum: 2026-06-02  
Marke: Nexyl Studio  

## Ziel

Dieses Template wird der erste verkaufbare Onepager fuer Beauty/Salon-Kunden.

## Pflichtsektionen

1. Hero mit Studio-Namen, kurzer Positionierung und Termin-CTA.
2. Services mit 3-6 Angeboten.
3. Preise oder Ab-Preise.
4. Studio-Vertrauen: Erfahrung, Atmosphaere, Hygiene, Bewertungen.
5. Galerie/Ergebnisse, nur mit Freigabe.
6. Kontakt/Buchung mit Standort, WhatsApp/Telefon und Buchungslink.

## Designrichtung

- ruhig, hochwertig, scanbar
- keine ueberladenen Kartenwelten
- nicht zu verspielt
- Fotos oder hochwertige Bildflaechen spaeter ergaenzen
- klare Buttons und gut lesbare Preise

## Technische Richtung

Start als statischer HTML/CSS-Onepager. Spaeter kann daraus ein Next.js- oder Astro-Template werden, wenn Deployment und Wiederverwendung wichtiger werden.

## Akzeptanzkriterien

- funktioniert auf Handy ohne Textueberlauf
- CTA in erster Ansicht sichtbar
- Services und Preise in unter 10 Sekunden erfassbar
- keine riskanten Claims
- lokal testbar
- bereit fuer kundenspezifische Inhalte
