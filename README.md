# nexi-imperium

Lokales Konfigurations- und Skript-Repo fuer Nexis KI-Zentrale.

Dieses Repo ist fuer technische Konfiguration, Setup-Dokumente und harmlose Skripte gedacht.
Es ist nicht fuer sensible Inhalte gedacht.

## Darf hinein

- Setup-Skripte
- Docker-Compose-Dateien ohne Secrets
- technische Dokumentation
- Beispielkonfigurationen ohne echte Tokens

## Darf niemals hinein

- Passwoerter
- Tokens/API-Keys
- Recovery Keys
- Drehbuecher/Rohideen
- Heilungs-Rohwissen
- Finanzstrategien
- Memory-Rohdaten
